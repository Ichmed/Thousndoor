Inferno

Formerly Dreftwod, the largest Island in the the <span class="dead-link">Deadman's Archipelago</span>.

After burning down the old town, which was a hive of scum and villainy, <a href="/Thousndoor/World/Factions/Burning Church/Burning Church.html">Burning Church</a> used the land to create a suburban settlement with ocean views.

The name is a joke, referencing <a href="/Thousndoor/World/Places/Districts/Paradiso.html">Paradiso</a> another tropical island that is also part of <a href="/Thousndoor/World/Places/Thousndoor.html">Thousndoor</a>
